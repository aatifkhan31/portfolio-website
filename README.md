# Personal Portfolio Website

A responsive personal portfolio website built using HTML and CSS to showcase my profile, technical skills, projects, and contact information.

## About

This portfolio website was created as part of my web development learning journey. It serves as an online resume and project showcase, allowing visitors to learn more about my educational background, skills, and projects.

## Features

- Responsive design
- Simple and user-friendly interface
- About Me section
- Projects section
- Contact information
- GitHub and LinkedIn links
- Mobile-friendly layout using CSS Flexbox

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
portfolio-website/

├── Portfolio.html
```
